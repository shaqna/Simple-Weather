# Simple Weather
### Date created
Jan, 2024

### Desscription
Weather app is android application that can show some information of weather arround user or based on user location.

# Tech Stack
- Programming language
  - Kotlin
- OS System
  - Android
- Dependency injection
  - Koin
- Design pattern
  - MVVM
- Permission library
  - EasyPermission
- Location service
  - Google Maps Service
